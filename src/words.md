# אמצעי קישור — Hebrew Linking Words Reference

> Used by the Section Writer (Skill 4: Academic Language & Linking Words).
> Every paragraph of 3+ sentences must contain at least 2 linking words, varied and contextually correct.

## Linking Words by Category

### תוספת
**Function:** Addition — adding supporting evidence or parallel arguments.

| מילת קישור | Usage Context |
|---|---|
| ו... וכן | Basic addition within a sentence |
| גם | Adding a similar point |
| בנוסף | Formal addition — opening a new supporting claim |
| נוסף על כך | Stronger formal addition |
| יתרה מכך | Escalation — "moreover", the next point is stronger |
| כמו כן | Parallel addition — "likewise" |
| לרבות | Inclusive listing — "including" |
| מלבד | "Besides", "apart from" |
| מעבר לכך | "Beyond that" — introducing an additional dimension |
| עוד ניתן לטעון כי... | Opening a new argumentative claim |

---

### ניגוד
**Function:** Contrast — presenting opposing views or counter-arguments.

| מילת קישור | Usage Context |
|---|---|
| אבל | Basic contrast (less formal) |
| לעומת / לעומת זאת | Direct comparison — "in contrast" |
| עם זאת | Concessive contrast — "nevertheless" |
| אך | Formal "but" |
| אולם | Formal "however" — opening a counter-argument |
| ברם | Literary/elevated "however" |
| אלא ש... | "Rather" — correcting a prior assumption |
| ואילו | "Whereas" — parallel contrast |
| לחילופין | "Alternatively" |
| אדרבא | "On the contrary" — strong reversal |
| בו בזמן ש... | "While" — simultaneous contrast |
| כנגד | "Against" — opposition |
| מצד אחד...מצד אחר | "On one hand...on the other" |
| מחד גיסא...מאידך גיסא | Formal version of "on one hand...on the other" |
| לכאורה...אך למעשה | "Seemingly...but in fact" — revealing hidden truth |
| אמנם...אבל | "Granted...but" — concession then counter |

### מטרה ותכלית
**Function:** Purpose — explaining goals and intentions.

| מילת קישור | Usage Context |
|---|---|
| כדי ש... | "In order that" — purpose clause |
| בשביל ש... | Less formal purpose (avoid in academic writing) |
| לשם כך | "To that end" — referring back to a stated goal |
| למען | "For the sake of" — elevated register |
| במטרה ל... | "With the aim of" — formal purpose |
| על מנת | "In order to" — standard academic purpose |
| לבל | "Lest" — negative purpose (literary) |
| פן | "Lest" — archaic/biblical register |
| שמא | "Lest perhaps" — expressing concern |

## Usage Guidelines for Section Writers

1. **Minimum density:** Every paragraph of 3+ sentences must contain **at least 2** linking words.
2. **Variety:** Do not repeat the same linking word within a paragraph. Vary across the category.
3. **Context match:** Choose the linking word that matches the logical relationship, not just any word from the list.
4. **Register:** Prefer formal/academic linking words (אולם, לפיכך, יתרה מכך) over colloquial ones (אבל, בגלל).
5. **Category awareness:** If a paragraph presents a counter-argument, use ניגוד words. If it draws a conclusion, use תוצאה words.
