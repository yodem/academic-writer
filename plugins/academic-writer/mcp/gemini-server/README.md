# academic-writer Gemini MCP server

Internal MCP server bundled inside the academic-writer Claude Code plugin. Routes the plugin's reader-facing prose phases — section drafting + 6 in-pipeline quality checks, cross-section synthesis, free-form edits, and voice calibration samples — to Gemini, while Claude retains orchestration and the citation audit. Claude Code spawns this server automatically when the plugin is enabled (declared under `mcpServers.gemini-api` in the plugin manifest); subagents inherit access to its five tools (`gemini_write_section`, `gemini_synthesize`, `gemini_edit`, `gemini_calibrate_sample`, `gemini_raw`). Auth, calibration-bundle caching, retry, and the 2.5 / 3.x thinking-API split are handled internally. Setup is owned by the plugin's `/academic-writer:setup` skill.
